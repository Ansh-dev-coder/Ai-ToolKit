# Ai-ToolKit
A full-stack AI toolkit built with React, Spring Boot, and OpenAI API featuring an AI Chatbot, AI Image Generator, and AI Recipe Generator.

## Modules
- 🤖 AI Chatbot
- 🖼️ AI Image Generator
- 🍳 AI Recipe Generator

## Tech Stack
- React
- Spring Boot
- OpenAI API
- Maven
- REST API
